中国队38金32银18铜收官 金牌数平境外奥运最佳

　　

8月8日，随着李倩在女子拳击中量级项目上拿下1银，中国奥运代表团在东京奥运会的征程也全部结束。在东京赛场上，中国健儿获得了38金32银18铜的成绩，金牌数追平2012年伦敦奥运会，平了海外参加奥运会的最佳战绩。从奖牌来看，创造了海外参加奥运会的最佳战绩。

　　

九年前的伦敦奥运会，中国健儿获得了38金、27银、23铜，奖牌总数88枚。后来，受兴奋剂复检的影响，中国在伦敦奥运会的奖牌最终为38金31银22铜，补上了4银，少了1铜，最终的奖牌数是91枚。

　　

九年后的东京，中国健儿获得了38金、32银、18铜，奖牌总数88枚。32枚银牌，也是中国在历届奥运会上获得银牌数最多的一届。 伦敦奥运会奖牌截屏伦敦奥运会奖牌截屏

　　

本次奥运会，中国的38金分布为：体操3金、田径2金、羽毛球2金、皮划艇1金、场地自行车1金、跳水7金、击剑1金、赛艇1金、帆船帆板1金、射击4金、游泳3金、乒乓球4金、蹦床1金、举重7金。中国队在14个大项上夺得了38枚金牌。 　　举重、跳水两大梦之队各自夺得7金，乒乓球、射击各自取得4金，四支队伍就拿到了22金，展现出了王牌项目的实力。田径女子田赛，巩立姣和刘诗颖夺得金牌，这两个项目都是首次获得奥运会的金牌。体操拿下3金，一扫里约奥运会0金阴霾，重回体操强国行列。 　　羽毛球队在这个奥运周期经历了新老交替，但队伍在东京展现出了不俗的竞技状态，收获2金4银。游泳队在没有了孙杨压阵的情况下，拿下3金，足够惊艳。赛艇、皮划艇项目时隔13年再夺奥运金牌，帆船帆板、蹦床、击剑时隔九年再夺奥运会金牌，场地自行车钟天使卫冕，这次搭档换成了鲍珊菊。 　　老将依然坚挺，00后甚至是05后冠军强势来袭，中国奥运代表团战东京收获满满。
